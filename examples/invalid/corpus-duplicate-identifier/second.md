# ADR-917: Second claimant of a shared identifier

## Status

Accepted

## Context

This artifact re-declares an id already claimed by another file.

## Decision

Collide on purpose, to demonstrate the finding.

## Consequences

References to this id are unresolvable until one artifact is re-identified.
